# Component Library Prompt

## Stack

- Vite
- React
- CSS Modules
- PostCSS (auto-prefixer)
- Typescript
- Storybook
- ViTest
- esLint

## Instructions

### Install

Clone Repo, then:

```
$ npm install
```

### Storybook

To run Storybook:

```
$ npm run storybook
```

It should open a browser window with the storybook

### Build Library

The build process uses the Vite library mode which packages everything up nicely.

```
$ npm run build
```
